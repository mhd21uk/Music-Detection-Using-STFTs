*Files in this folder:

- STFT_Music_Recognition_Final.ipynb --> Python notebook where all the code is
- Music_Recognition_Results_Visualisation.ipynb --> Python notebook just for results visualisations (manually done)

- Extreme_Hash_Table.pkl --> Hash table object with fingerprints of all extremist songs
- Pop_Extreme_Hash_Table.pkl --> Hash table object with fingerprints of some extremist songs
- Pop_Hash_Table.pkl --> Hash table object with fingerprints of 11 pop songs

- Extreme_database_songs_df --> Dataframe object with songs trained on
- Pop_extreme_database_songs_df --> Dataframe object with songs trained on
- Pop_database_songs_df --> Dataframe object with songs trained on

- whole_clips_results_df --> Dataframe of whole clip test results
- *.csv files --> result file for each of the three tests performed

*Folders in this folder:

- Pop songs --> Wav files of pop songs used
- Test Pop Songs --> wav files of pop songs' snippets (with added noise) for testing
- Test Pop Extremist Songs --> wav files of pop/extreme songs' snippets (with added noise) for testing
- Test Complete Audio Clips --> wav files of raw complete audio clips for testing

*Instructions: The layout of the code along with comments and instructions are found inside the python jupyter notebook.
